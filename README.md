# WAD — Wrapping Art Design

This is the website for Wrapping Art Design (WAD), a car detailing company.

## Making the repository and website public

### 1. Make the GitHub repository public
1. Go to the repository on GitHub: **Settings → General → Danger Zone → Change visibility**
2. Select **"Change to public"** and confirm.

### 2. Enable GitHub Pages (publish the website)
1. Go to **Settings → Pages** in your repository.
2. Under **Source**, select **GitHub Actions**.
3. Push a commit to the `main` branch — the workflow in `.github/workflows/deploy.yml` will automatically build and publish the site.
4. Your site will be live at: `https://<your-username>.github.io/WAD/`
